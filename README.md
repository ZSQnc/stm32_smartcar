
# STM32多功能智能小车 | 已调试成品版
（BT04-A蓝牙+PID精准循迹+红外避障+红外遥控+OLED实时显示）

## 项目简介
本项目基于**STM32F103C8T6**主控开发，是一套完整落地、软硬件联调成功的嵌入式智能小车项目。
全程解决前期核心问题：优化TIM4定时器PWM引脚与OLED引脚冲突，蓝牙标配**BT04-A透传模块**，所有功能实测跑通、参数校准完毕，可直接作为：本科课设/毕设、电子竞赛基础样机、GitHub简历硬核实战项目。

## 一、核心硬件清单（实物成品）
- 主控：STM32F103C8T6 最小系统板
- 蓝牙通信：BT04-A 主从一体串口蓝牙透传模块
- 循迹感知：多路红外黑白线循迹传感器阵列
- 避障模块：红外距离检测传感器（阈值已现场校准）
- 显示模块：IIC-OLED显示屏（独立优化引脚，无资源冲突）
- 动力驱动：L298N电机驱动板 + 直流减速电机
- 供电系统：锂电池组+充电宝，L289N驱动板自带5V稳压,续航稳定

## 二、已实测落地·全部可用功能
✅【BT04-A蓝牙控制｜实测可用】
手机蓝牙串口APP直接搜索连接BT04-A，下发指令：前进/后退/左转/右转/原地转向/停车，响应零延迟，串口通信稳定不丢包。

✅【PID闭环循迹｜校准完成】
采用增量式PID算法，现场实地调参固化最优参数；直线稳、转弯顺滑、急弯不脱线、黑线识别精准，复杂赛道连续跑通无偏移。

✅【智能红外避障｜优先级最高】
行车过程实时探测前方障碍，触发后**立即打断循迹/手动模式**，自动停车+后退避让，规避碰撞，安全逻辑闭环。

✅【红外离线遥控｜独立可用】
无手机、无蓝牙场景下，配套红外遥控器可全程手动操控，按键响应灵敏，适配现场演示、离线调试。

✅【OLED实时屏显｜全程可视化】
上电即亮，实时刷新展示：
蓝牙连接状态 / 当前运行模式（自动PID/手动遥控/蓝牙）/ 传感器实时数值 / 电机PWM档位 / 故障提示

✅【引脚彻底优化｜硬件无冲突】
原版TIM4（PB6/PB7/PB8/PB9）电机PWM与OLED引脚复用冲突，已全程修改底层驱动+重新分配硬件引脚，所有外设同时工作互不干扰，长期运行不死机、不蓝屏。

## 三、系统工作逻辑（成品实测流程）
1. 上电初始化：所有传感器、BT04-A蓝牙、OLED、电机PWM全部自检完成；
2. 默认进入**自动PID循迹模式**，小车沿黑线自主行驶；
3. 避障传感器检测到障碍物 → 立即高优先级触发避让保护；
4. 手机连BT04-A蓝牙 / 按下红外遥控 → 一键切入手动控制，覆盖自动循迹；
5. 所有运行状态、传感器数据同步实时投屏到OLED屏幕，全程可视。

## 四、模块成品实测说明
### 1. BT04-A蓝牙模块（已适配调试）
- 通信方式：串口USART透传，默认波特率匹配模块出厂参数；
- 使用：无需AT配置，开机即可被蓝牙APP搜索配对；
- 实测：连续长时间连接不断线，指令解析精准，无延迟乱码。

### 2. PID循迹算法（现场固化参数）
已多次实地跑赛道调参，平衡灵敏度与稳定性，解决新手常见：跑偏、冲出黑线、转弯卡顿等问题，成品直接可用。

### 3. 避障+红外遥控
障碍触发阈值现场标定，适配室内常规环境；红外遥控按键全映射实测，离线演示完全独立可用。

### 4. OLED显示
引脚彻底隔离优化，跟电机PWM、蓝牙串口互不抢占资源，上电秒亮，数据刷新流畅稳定。
