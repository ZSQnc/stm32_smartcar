# STM32智能小车 – 多功能集成（循迹+避障+红外遥控+蓝牙控制+OLED显示）

## 项目简介

本项目基于 **STM32F103** 主控芯片，使用 **标准外设库（SPL）** 开发，实现了一款多功能智能小车。支持三种工作模式，通过按键循环切换：

- **模式1**：PID循迹 + 红外避障（OLED不显示，保证实时性）
- **模式2**：红外遥控（OLED显示运动状态和PWM）
- **模式3**：蓝牙控制（支持速度调节，OLED显示速度）

小车具备完善的人机交互：OLED显示、蜂鸣器提示、数码管显示当前模式。

## 功能说明

### 模式1：PID循迹 + 红外避障
- 小车沿黑线行驶，使用**位置式PID**控制左右轮差速，循迹平稳。
- 当任意避障传感器检测到障碍物时，蜂鸣器长鸣报警，小车执行：**停止 → 后退 → 转向（左侧障碍右转，右侧障碍左转）→ 前进**，然后继续循迹。
- OLED 不显示（保证实时性），数码管显示“1”。

### 模式2：红外遥控
- 使用 MP3 遥控器控制小车运动，键值如下（已验证）：

| 按键 | 功能 | 32位键码（十六进制） |
|------|------|---------------------|
| 前进（CH-） | 持续前进 | 0x007FB14E |
| 后退（CH+） | 持续后退 | 0x007FD42B |
| 左转（PREV） | 左转400ms后停止 | 0x007F916E |
| 右转（NEXT） | 右转400ms后停止 | 0x007FE11E |
| 停止（PLAY/PAUSE） | 立即停止 | 0x007F817E |

- 按下任意有效键，蜂鸣器短鸣一声。
- OLED 显示当前模式、运动状态（RUN/BACK/LEFT/RGHT/STOP）以及左右轮PWM值。
- 数码管显示“2”。

### 模式3：蓝牙控制
- 手机安装蓝牙串口助手（如“蓝牙调试器”），连接蓝牙模块（BT-04A 兼容，波特率 9600）。
- 发送单字符指令控制小车：

| 指令 | 功能 |
|------|------|
| `F` | 持续前进 |
| `B` | 持续后退 |
| `L` | 左转400ms |
| `R` | 右转400ms |
| `S` | 停止 |
| `+` | 速度 +10（上限100） |
| `-` | 速度 -10（下限20） |

- 每次收到指令蜂鸣器短鸣一声。
- OLED 显示模式、运动状态、PWM值以及当前速度（例如 `SPD:080`）。
- 数码管显示“3”。

### 全局功能
- 短按按键（PA15）循环切换模式：1 → 2 → 3 → 1...
- 切换时蜂鸣器双短促音，OLED清屏并显示新模式界面。
- 数码管同步显示模式数字。

## 使用说明

### 1. 编译与下载
- 使用 Keil MDK 5 打开工程，确保已安装 STM32F10x 器件支持包。
- 编译（F7）生成 `.hex` 文件。
- 通过 ST-Link 或串口（FlyMcu）下载到开发板。

### 2. 上电启动
- 连接电池，开发板电源指示灯亮。
- OLED 显示开机画面“STM32 Car / Press Key / to Start”，蜂鸣器响两声。
- 短按按键，小车进入模式1。

### 3. 模式切换
- 每次短按按键循环切换模式，蜂鸣器双短促音提示，OLED 刷新。

### 4. 红外遥控
- 将红外接收头对准遥控器，在模式2下按遥控器按键即可控制。
- 若按键无反应，请检查红外接收头接线（OUT→PA8，GND→GND，VCC→3.3V）和遥控器电池。

### 5. 蓝牙控制
- 手机打开蓝牙，搜索设备（名称通常为 `BT-04A`），配对密码 `1234`。
- 打开蓝牙串口助手，连接设备，发送指令（注意发送格式为 ASCII 字符，不要选 HEX）。
- 在模式3下发送 `F`、`B`、`L`、`R`、`S`、`+`、`-` 控制小车。

### 6. 循迹与避障
- 模式1下，将小车放在黑线轨迹上，会自动循迹。
- 用手遮挡避障传感器，小车应后退并转向避开。

## 注意事项

1. **供电**：使用4节全新南孚5号电池（6V）或2节18650锂电池（7.4V）。电压低于5V时电机无力，低于3.5V时STM32可能复位。
2. **红外循迹传感器**：需要调节模块上的电位器，使黑线时指示灯亮、白地时灭。左右两个传感器分别调节。
3. **红外避障传感器**：同样调节电位器，使检测距离约5-10cm。
4. **OLED显示**：模式1下OLED不刷新，此为正常设计，避免干扰循迹实时性。
5. **蓝牙模块**：若连接不上，可先取消手机系统蓝牙中的配对，重新用APP搜索连接。

## 红外遥控器码值表（厂家提供与实际测试对照）

厂家提供的码值表如下（24位码），实际程序中使用了32位码（高8位为0x00，中8位为0x7F，低16位为按键码）：

| 按键 | 厂家24位码 | 程序中的32位码 |
|------|-----------|----------------|
| CH-（前进） | FFA25D | 0x007FB14E |
| CH+（后退） | FF629D | 0x007FD42B |
| PREV（左转） | FFE21D | 0x007F916E |
| NEXT（右转） | FF22DD | 0x007FE11E |
| PLAY（停止） | FFC23D | 0x007F817E |

> 注：程序中的32位码已通过实际遥控器测试得出，与厂家提供的24位码不完全相同，以实际测试为准。
